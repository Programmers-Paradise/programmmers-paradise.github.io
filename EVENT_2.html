About the Event

Dear students,

We are pleased to announce the Member Selection Test on 18th MAY 2024 for this semester.

  • Total number of seats available: 80
  • Number of seats reserved for existing members: 38
  • Seats remaining for this intake: 42

Breakdown of seats:

  • 12 seats allocated for 6th-semester students.
  • 20 seats allocated for 4th-semester students.
  • 10 seats allocated for 2nd-semester students.

Key points:

  • There will be different sets of questions tailored for each semester’s curriculum.
  • The leaderboard will be determined using the ICPC method.
  • Timing for the test: 5:00 PM to 6:30 PM.
  • It is mandatory for all participants to be part of the “Holidays with Code” competition.
  • Results of the selection test will be announced the following day.
  • There will be NO further recruitment of new members this semester.
